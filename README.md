## `// about`

Full-stack & AI engineer who likes clean systems and things that actually ship. I build cross-platform apps with **Flutter**, back-ends with **Python / FastAPI + PostgreSQL**, and interfaces with **React / TypeScript** — with a bias toward **LLM tooling** and **trading systems**.
